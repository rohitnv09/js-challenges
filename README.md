# JavaScript Challenges

A collection of JavaScript challenge solutions, experiments, and small problem-solving exercises.

This repo is intentionally simple: mostly `.js` files, each one focused on a specific challenge or concept.

## What You'll Find

- JavaScript language practice
- Problem-solving exercises
- Small utility implementations
- Edge cases and reasoning-heavy examples
- Solutions that explore different approaches

## How To Use

- Open a challenge file
- Try solving it yourself first
- Compare your solution with the one in the repo
- Add edge cases and tests where useful
- Refactor for clarity and correctness

## Learning Focus

The goal of this repo is not just to get the answer, but to understand:

- why a solution works
- what edge cases matter
- how JavaScript behaves under the hood
- how to write cleaner and more reliable code

## Contributing

Feel free to add new challenge files or improve existing ones.

A good contribution usually includes:

- a clear challenge
- a working solution
- comments for tricky parts
- edge cases when relevant
